1. Go to dashboard.stripe.com/products → create two products ($15 day, $75 week)
2. Click each product → Buy button tab → add "License plate" custom field → Get code
3. Copy the 2-line embed code Stripe gives you
4. Replace the blue dashed placeholder boxes above with your Stripe code
5. Done — payments go directly into your Stripe account
